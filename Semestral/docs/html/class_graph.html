void Graph::addRelation ( int file,
int dep )

Přidá relaci (hranu) mezi dvěma soubory do grafu.

Vytváří dvě hrany:

  1. V původním grafu: file -> dep (file závisí na dep)
  2. V transponovaném grafu: dep -> file (reverse direction)
Parametry
fileSoubor, který závisí (zdrojový soubor).
depSoubor, na kterém file závisí (cílový soubor).

◆ bfs()

std::set< Node * > Graph::bfs ( int node)

Provádí BFS na transponovaném grafu.

Provádí prohledávání do šířky (BFS) na transponovaném grafu.

Metoda najde všechny uzly dosažitelné z daného uzlu v transponovaném grafu.

Algoritmus:

  1. Najde se startovací uzel podle zadaného ID v transponovaném grafu
  2. Provede se BFS z tohoto uzlu
  3. Vrátí se množina všech dosažených uzlů bez duplicit

Časová složitost: O(V + E), kde V je počet vrcholů a E je počet hran. Prostorová složitost: O(V) pro frontu a výstupní množinu.

Parametry
nodeId uzlu, ze kterého se má začít vyhledávání.
Návratová hodnota
Množina ukazatelů na všechny uzly dosažitelné z počátečního uzlu v transponovaném grafu.
